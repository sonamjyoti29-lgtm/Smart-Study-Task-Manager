# Smart Study Task Manager (C++)

## 📌 Description
This is a console-based task management system built using C++.
It allows students to add, view, complete, and delete study tasks.

## 🛠 Features
- Add Task
- Show Tasks
- Mark Task as Completed
- Delete Task
- Menu Driven Program

## 📚 Concepts Used
- OOP (Class & Object)
- Vector (Dynamic Array)
- Loops & Switch Case
- Basic Searching

## 🎯 Learning Outcome
- Improved understanding of C++ and Data Structures
- Practical implementation of vector
- Hands-on experience with menu-driven programs
